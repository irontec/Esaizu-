 Descripción de Esaizu
 =====================
 Esaizu es una aplicación web libre que permite tener en un único lugar todas tus fuentes de información: twitter, facebook, linkedin, wordpress, y feeds.
 
 Programación de Esaizu
 ======================
 Esaizu ha sido escrita sobre PHP, MySQL, Jquery, CodeIgniter, Zend library y ha sido testado en el sistema operativo GNU/Linux.
